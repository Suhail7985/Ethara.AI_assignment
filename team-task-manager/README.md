# TaskFlow Workspace

Professional team management and project tracking suite.

## Features

- Role-based workspace access
- Real-time workflow visualization
- Integrated productivity analytics
- Secure authentication system
- Dark mode optimized interface

## Technology

- React / Node.js / Express
- MongoDB / Socket.io
- Tailwind CSS

## Deployment

### Environment Variables

#### Backend
- `MONGODB_URI`
- `JWT_SECRET`
- `CLIENT_URL`
- `PORT`

#### Frontend
- `VITE_API_URL`
